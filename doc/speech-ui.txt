Commands
--------

1. Overview

You can *always* say "What?" to repeat the last thing said by Gaia.

1.1 Short command reference

* "Hi Gaia" or "Command"
* "Goodbye"
* "Call X" or "Call contact X"
* "Hangup X" or "Hangup contact X"
* "Join X" or "Join group X" 
* "Leave X" or "Leave group X"





2. Dialogue

2.1 Entering and leaving Gaia command mode:

-> "Hi Gaia", "Command"
   [Enters gaia command mode]
   [A beep is played played when you enter command mode]
   [During command mode an ambient continous beep is played in the background]

-> "Goodbye"
   [Leaves gaia command mode]
   [A beep is played played when you leave command mode]
   NOTE: Can be used at any time when in command mode!!!

2.2 Call peer 

-> "Call X" or "Call contact X"
  <- "Do you want to call X?"
    -> "Yes" or "Yeah"
      <- "You are now in a call with  X"
        [Leaves command mode]
        NOTE: See "Incoming asynchronous messages" below
      <- "You are already in a call with X"
        [Leaves command mode]
    -> "No" or "Nah"
       <- "OK"
         [Leaves command mode]
  <- "X is not known. Please try again!"
     [Stays in command mode]

2.3 Hangup peer

-> "Hangup X" or "Hangup contact X"
  <- "Do you want hangup X?"
    -> "Yes" or "Yeah"
      <- "You are no longer in a call with X"
        [Leaves command mode]
        NOTE: See "Incoming asynchronous messages" below
      <- "You are not in a call with X"
        [Leaves command mode]
    -> "No" or "Nah"
       <- "OK"
         [Leaves command mode]
  <- "X is not known. Please try again!"
     [Stays in command mode]

2.4 Join group

-> "Join X" or "Join group X"
  <- "Do you want to join X?"
    -> "Yes" or "Yeah"
      <- "You are now an active member of X"
        [Leaves command mode]
        NOTE: See "Incoming asynchronous messages" below
      <- "You are already an active member of X"
        [Leaves command mode]
    -> "No" or "Nah"
       <- "OK"
         [Leaves command mode]
  <- "X is not known. Please try again!"
     [Stays in command mode]

2.5 Leave group

-> "Leave X" or "Leave group X"
  <- "Do you want leave X?"
    -> "Yes" or "Yeah"
      <- "You are no longer active in group X"
        [Leaves command mode]
        NOTE: See "Incoming asynchronous messages" below
      <- "You are not an active member of X"
        [Leaves command mode]
    -> "No" or "Nah"
       <- "OK"
         [Leaves command mode]
  <- "X is not known. Please try again!"
     [Stays in command mode]
