aplay
-----
$ arecord -f cd -d 5 -D bluealsa:DEV=20:74:CF:72:85:82,PROFILE=sco test.wav
$ aplay -D bluealsa:DEV=20:74:CF:72:85:82,PROFILE=a2dp test.wav

with this .asoundrc:

defaults.bluealsa.service "org.bluealsa"
defaults.bluealsa.device "20:74:CF:72:85:82"
defaults.bluealsa.profile "a2dp"
defaults.bluealsa.delay 10000

in that case this works as well:

$ arecord -f cd -d 5 -D bluealsa:PROFILE=sco test.wav
$ aplay -D bluealsa test.wav

.asoundrc could probably be written to be even better

Make sure that:

systemctl cat bluealsa
...
ExecStart=/usr/bin/bluealsa -p a2dp-sink -p a2dp-source -p hfp-ag -p hsp-ag
